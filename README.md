# data-transfer installation guide

```bash
# Get the project
$ git clone https://github.com/TECH4DX/data-transfer.git
# Run the project to read data from the OpenSearch
$ ./run.sh read
# Run the project to write data to the ElasticSearch
$ ./run.sh write
```
